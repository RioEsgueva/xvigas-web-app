Help for XVIGAS

XVIGAS is a free web app funded from adds. It allows to resolve exercises and problems beams. XVIGAS web app is based on the previous XVIGAS for Windows v0.2f. The main advantage of XVIGAS web app is that it works in every operating system, also it works in portable devices, such smartphones and electronic tablets. XVIGAS web app es una aplicación web gratuita financiada a partir de la publicidad que permite resolver vigas de ejercicicios y problemas. XVIGAS para la web está basado en la versión 0.2f del programa que ya existía para Windows. La principal ventaja de XVIGAS Web app es que funciona en cualquier sistema operativo, incluyendo dispositivos móviles como teléfonos inteligentes y tabletas electrónicas.

A HTML5, JavaScript & CSS3 web browser is required, almost every uptated web browser could work. Additional addons are not needed. Data is processed in the user's computer, data is not sent through Internet, the app can run offline. El requisito es tener un navegador web compatible con HTML5 y CSS3 (cualquier navegador actualizado podrá funcionar). No hacen falta instalar complementos adicionales. Los datos se procesan en el ordenador, no se envía información a través de Internet, con lo que la aplicación puede funcionar fuera de línea (offline).



Units

XVIGAS works with all units systems, all values must have the same units. If length is in inches, Young's modulus and inertia should have the same units. If length is in meters, height of supports should be also in meters.

Datos generales

The beam length should be a positive value.

If Young's modulus and inertia are unknown, they can be left empty, the beam will be calculated as 1 for these values.

There are some tables with some examples of values for these fields. There is also a inertia calculator for rectangular and elliptical sections.

The inertia calculator can provide values for solid sections (use e = 0), or hollow sections. Hollow sections can be calculated as thin wall or thick wall. If the thickness value is negative, the inertia will be calculated as thin wall section (e >> a, b), otherwise (positive values), It will be calculated as a thick wall section.

Beam types

There are various beam types with different configurations of supports, cantilevers and pins. In composed beams (d, e, f y g), a second subtype should be chosen in the next list.

Supports

El recuadro de los apoyos es diferente según el tipo de viga o incluso puede no estar presente (vigas sin apoyos).

Isostatic beams (simple or composed)

En las vigas que tienen un número fijo de apoyos y rótulas aparece una lista con entradas de texto donde se introducen los datos de estos elementos. Se debe respetar el orden de los apoyos. Los apoyos pueden tener un desplazamiento vertical, que es positivo si es hacia arriba. Este desplazamiento no es tenido en cuenta en el cálculo de fuerzas, sólo en las deformaciones.

Hyperestatic beams

Los tipos de vigas con un número de apoyos variable (las hiperestáticas) tienen un diálogo diferente, donde primero se introduce la posición y el descenso de un apoyo y se pulsa en la marca verde para introducirlo en la lista inferior, pudiéndose añadir tantos apoyos como sean necearios. Para eliminar los apoyos hay que pulsar en la marca roja de la lista que corresponda al apoyo a eliminar.

Loads

Las cargas se introducen de forma similar a los apoyos en vigas hiperestáticas, primero se introduce la posición y la magnitud y luego se pulsa la marca verde para introducir cada carga. Se pueden eliminar cargas de la lista haciendo clic en el botón rojo para eliminar. Las cargas introducidas se actualizan en el esquema de cargas y elementos.

If a new load doesn't appear in the list when it is added, it could have wrong values. Position must be higer than 0 and lower than beam lenght. Magnitude should not be zero, but a variable distributed load could have this value if booht .

Results

Finally the button Results does the beam analysis and shows the results. Caution: data validation is not implemented yet, if not valid values are entered (two supports in the same position, for example) the results could be wrong.

La escala de los gráficos se toma del mayor valor presente en la viga, de forma que siempre se ocupa el máximo espacio disponible. Hay que tener cuidado de que una viga con unas cargas diferentes puede parecer que tiene el mismo nivel de deformación que otra con menos cargas si no se tiene en cuenta que las vigas usan escalas diferentes.

At the end of the results list, there are a entry for get the values (shear, moment, slope and displacement) on a certain point of the beam. By default, the values are taken on the left side if they are different.

Additional notes

Results can be different than XVIGAS for Windows, because calculations are done with 64 bits floating point values. Win32 app uses 32 bits floats.
