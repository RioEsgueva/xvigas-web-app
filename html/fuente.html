Información sobre XVIGAS

XVIGAS está disponible bajo la licencia GPL v3.

El codigo puede ser examinado directamente en la aplicación, ya que actualmente no está comprimido.
También está disponible un repositorio de XVIGAS en GitHub.

La aplicación está basada en XVIGAS para Windows, una aplicación que empecé a escribir en 2004, primero era una aplicación para consola, posteriormente la programé con interfaz gráfica. Finalmente decidí hace poco portarla a HTML+CSS+JavaScript para que pudiera funcionar en la web y así esté disponible para cualquier sistema operativo y dispositivo. En XVIGAS web app he intentado mantener la funcionalidad simplificando la interfaz.
